# 🧠 AI Fitness Agent

An intelligent LLM-powered fitness assistant that generates personalized workouts, diet plans, and insights using memory, RAG, and adaptive logic.

## 🚀 Features

- 🏋️ Workout generation (focus + intensity aware)
- 🥗 Diet planning (goal + preference based)
- 📊 Progress tracking & insights
- 🔔 Smart nudges
- 🧠 Memory-aware personalization
- 🔍 RAG-based exercise retrieval
- ⚡ Adaptive workout modification

## 🧠 Tech Stack

- FastAPI
- Groq (LLM)
- SQLite
- Streamlit
- Python

## 🏗️ Architecture

User → LLM Agent → Tool Selection → Execution → Adaptive Layer → Response

## 🔥 Highlights

- Built custom LLM agent framework (no LangChain)
- Robust JSON parsing & guardrails
- Context-aware decision making
